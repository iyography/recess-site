## QUESTION TYPES LEGEND

- **Standard Question**: Single selection, auto-advances to next question
- **MULTI-SELECT**: User can select multiple options (1-2, 1-3, or 1-4 as specified), requires "Next" button to advance
- All questions include hover states, selection highlighting, and progress tracking
